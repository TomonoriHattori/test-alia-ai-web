Introduction

Digital transformation of companies to the next stage with AI. Secure DX app "ailia DX Insight"
"ailia DX Insight" for Windows and macOS. It is a functioning DX app.
In addition to regular ChatGPT functions ailia's unique text search and image search.
Adding functions such as voice recognition.

"ailia DX Insight" is safe.

Previously, in order to handle text within a company using ChatGPT, it was necessary to upload all the text to a server. With "ailia DX Insight", however, text within a company is processed offline and only a portion of the text found through searches is uploaded to the server. Additionally, for applications that require a high level of security, it is equipped with the capability to use a local LLM instead of ChatGPT. By using a local LLM, it is possible to build a completely secure environment.

FEATURE

Just install it.
Various AI are available for use.
From sales positions to research roles and even comedy,
"ailia DX Insight" supports the DX of a wide range of tasks with a single, easy-to-use application.

  • Article search modeUnique function

    ChatGPT can only provide answers based on publicly available information, so it cannot answer questions about internal company information or product information. However, with "ailia DX Insight", it is possible to use PDFs or text to provide ChatGPT with answers based on internal company information. Additionally, it can answer questions in Japanese regarding English specification documents and can be effective in providing user support for products.

  • Image and Audio search modeUnique features

    With "ailia DX Insight", you can search images and audio offline from text. AI automatically tags images and audio, allowing you to search for photos and audio within a company using text. It also supports searching for similar files from images and audio.

  • Image generation modeUnique function

    With "ailia DX Insight", you can generate images for presentations, etc. You can use DALL·E to generate images from text.

  • Minutes creation modeUnique functionality

    ailia DX has a high-performance offline speech recognition engine, which allows you to securely obtain meeting minutes within a company offline. You can also create meeting minutes from a computer's microphone.

  • Automate tasks with prompts
    Unique functionality

    Workflows using ChatGPT can be executed with prompts. You can select multiple files and apply prompts, making it possible to do tasks like contract comparisons.

  • ChatGPT and Gemini
    Compatible with both

    Supports standard features of ChatGPT and Gemini.
    With the comfortable operability of a native app, it works more smoothly than using ChatGPT or Gemini from a browser.

  • Support for local LLM

    It is compatible with local LLMs, such as Gemma2. By using a local LLM, it is possible to operate "ailia DX Insight" in a completely secure environment without using ChatGPT.

Frequently Asked Questions

Frequently Asked Questions
Regarding the frequently asked questions inquiries,
We will post the answer.
  • QPlease let me know the specifications of the available computer.
    AYou need a CPU of Core i7 or higher and 8GB or more of memory in Windows environments.
    For macOS environment, a CPU of M1 or higher and 8GB or more of memory are required.
  • QDo you need a GPU?
    Aailia DX Insight can be executed using only a CPU.
    When using a compatible GPU (such as the built-in GPU of the Intel® Core™ Ultra series or the Intel® Arc™ GPU), high-speed processing is possible for document registration and minute creation.
  • QCan I use the shared folder?
    Aailia DX Insight can handle mounted files, so it can also use shared folders via the network.
  • QPlease tell me the supported file formats.
    AIt supports the following document formats.
    'c', 'cpp', 'csv', 'json', 'java', 'md', 'py', 'rb', 'tex', 'txt', 'css', 'php', 'pdf', 'docx', 'xlsx'

    It supports the following image formats.
    'png', 'jpg', 'jpeg'

    It supports the following audio formats.
    'wav'
  • QCan it be used without a ChatGPT API KEY?
    Aailia DX Insight supports ChatGPT, Gemini, Azure, OpenAI-compatible servers, and local LLM.
    When using a local LLM, it can be used without a ChatGPT API KEY.
    Additionally, since meeting minutes and image searches, as well as OCR, are always processed locally, they can be used without a ChatGPT API KEY.
  • QPlease let me know about the functions you plan to add in the future.
    AWe plan to add a dark mode and index management functionality.
  • QPlease tell me the price.
    AThe evaluation version can be used for free for 30 days.
    The full version is offered on a subscription model.
    Please contact our sales team for pricing.
